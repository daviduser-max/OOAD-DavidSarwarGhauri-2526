Statische properties

Doel: statische properties maken die moeten geïmplementeerd worden in afgeleide klassen.

Theorie

Een statische property hoort bij de klasse, niet bij een instantie. Ze wordt gedeeld over alle objecten en aangesproken via de klassenaam. 

public class Teller
{
   public static int AantalAangemaakt { get; private set; }
   public string Naam { get; set; }

   public Teller(string naam)
   {
      Naam = naam;
      AantalAangemaakt++; // verhoog de gemeenschappelijke teller
   }
}

Teller t1 = new Teller("Alice");
Teller t2 = new Teller("Bob");
Console.WriteLine(Teller.AantalAangemaakt); // 2 — via klassenaam 'Teller'

Opgave

Gegeven is de klasse WorkshopDeelnemer in Exercises/Classes met properties Naam en IsAanwezig (private setters) en een constructor met parameters voor naam en aanwezigheidsstatus.

  1. Voeg twee static properties toe:
    • AantalAangemaakt (private setter)
    • AantalAanwezig (private setter)
  2. Pas de constructor aan zodat beide statische properties correct bijgehouden worden bij elke aanmaak.
  3. Voeg een instantiemethode ZetAfwezig() toe die de aanwezigheidsstatus aanpast en AantalAanwezig correct bijhoudt. Let op: als een deelnemer al afwezig is, mag het teller niet nog eens verlaagd worden.

Verwacht resultaat:

-------------

Aantal aanwezig: 2
Aantal aanwezig na wijziging: 1

Statische methodes

Doel: statische methodes gebruiken.

Theorie

Een statische methode roep je aan via de klassenaam, zonder een object aan te maken. Ze heeft enkel toegang tot statische properties, niet object properties. 

public class Validator
{
   // static methode: aanroepen via Validator.IsGeldigEmail(...)
   public static bool IsGeldigEmail(string email)
   {
      return email.Contains("@") && email.Contains(".");
   }
}

bool ok = Validator.IsGeldigEmail("test@mail.be"); // methode opgeroepen rechtstreeks vanop de klasse

Met Regex.IsMatch() controleer je een string tegen een reguliere expressie (patroon). Enkele veelgebruikte symbolen: 

// [A-Z]   → één hoofdletter
// \d      → één cijfer
// {3}     → exact 3 keer
// ^...$   → match volledig (begin tot einde)

string patroon = @"^[A-Z]{3}\d{2}$"; // bv. "ABC12"
bool geldig = Regex.IsMatch("ABC12", patroon); // true
bool ongeldig = Regex.IsMatch("ab12", patroon); // false

Opgave

  1. Maak in Exercises/Classes een klasse CouponCode:
    • Private static veld: _couponRegex = @"^[A-Z]{3}\d{2}-[A-Z]{2}$"
    • Property Code (string)
    • Berekende property IsGeldig (bool, alleen getter): gebruik Regex.IsMatch() om te controleren of Code aan het patroon voldoet
    • Constructor met één parameter
  2. Voeg een static methode ControleerCode(string code) toe die true of false teruggeeft.
  3. Voeg een static methode Beschrijf(string code) toe:
    • Als de code geldig is: "Prefix=ABC, Nummer=12, Regio=DE"
    • Als de code ongeldig is: "ongeldige code"

Voorbeeldcodes om te testen: ABC12-DE (geldig), AB12-DE (ongeldig), XYZ99-BE (geldig).

Statische klassen

Doel: statische klasse met alleen static members gebruiken.

Theorie

Een statische klasse bevat alleen statische methodes en properties. Je kan er geen objecten uit maken (en er zijn dus bijgevolg ook geen constructors). Ze dienen vooral als een verzameling hulpmethodes en properties (utility class). 

// static klasse
public static class MathHelper
{
   public static int Kwadraat(int n) => n * n;
   public static bool IsPositief(int n) => n > 0;
}

Console.WriteLine(MathHelper.Kwadraat(4)); // 16
MathHelper mh = new MathHelper(); // fout! kan geen objecten maken uit een statische klasse

Opgave

  1. Maak in Exercises/Classes een statische klasse TekstAnalyse met:
    • Privaat static veld verbodenWoorden: array met "delete", "drop", "truncate"
    • Privaat static veld verbodenKarakters: array met '!', '@', '#', '$', '%'
  2. Voeg deze static methodes toe:
    • AantalWoorden(string tekst): telt woorden op basis van spaties
    • BevatVerbodenWoord(string tekst): geeft true als de tekst een verboden woord bevat
    • BevatVerbodenKarakter(string tekst): geeft true als de tekst een verboden karakter bevat
    • IsGeschiktVoorTitel(string tekst): geeft true als:
      • de tekst niet leeg is
      • de tekst minimaal 5 en maximaal 30 tekens lang is
      • de tekst geen verboden woorden of karakters bevat
  3. Zorg dat alle methodes null- en lege-string-veilig zijn.

Bestelstatus

Doel: een enum definiëren en gebruiken als type van een property.

Theorie

Een enum is een type met een vaste set benoemde waarden. Het maakt code leesbaarder dan losse getallen of strings. 

public enum SeizoenType
{
   Lente,
   Zomer,
   Herfst,
   Winter
}

Je gebruikt een enum zoals elk ander gegevenstype, b.v. voor een property in een klasse:

public class Activiteit
{
   public string Naam { get; set; }
   public SeizoenType BesteSeizoen { get; set; }
}

Activiteit a = new Activiteit { Naam = "Zwemmen", BesteSeizoen = SeizoenType.Zomer };

if (a.BesteSeizoen == SeizoenType.Zomer)
{
   Console.WriteLine("Ideaal in de zomer!");
}

Opgave

  1. Maak in Exercises/Enums een enum BestelStatus met de waarden: Nieuw, InBehandeling, Verzonden, Geleverd, Geannuleerd.
  2. Maak in Exercises/Classes een klasse Bestelling met properties: KlantNaam, ProductNaam en Status (van type BestelStatus).
  3. Voeg een berekende property KanNogGewijzigdWorden toe die true geeft zolang de bestelling nog wijzigbaar is (niet Geleverd of Geannuleerd).
  4. Voeg een override ToString() toe die klantnaam, productnaam en status netjes weergeeft.

Verwacht resultaat:

Oefening 4: bestelstatus
-------------
Laptop - Nieuw - wijzigbaar: ja
Muis - Verzonden - wijzigbaar: nee

Enum conversies

Doel: een enum omzetten naar int en omgekeerd.

Theorie

Elke enum-waarde heeft intern een int-waarde (standaard 0, 1, 2, …). Je kan heen en weer casten: 

public enum SeizoenType { Lente, Zomer, Herfst, Winter } // Lente = 0, Zomer = 1...
SeizoenType s = SeizoenType.Herfst; // 2

int getal = (int)s;                // enum → int: 2
SeizoenType s2 = (SeizoenType)3;   // int → enum: Winter
string tekst = s.ToString();       // enum → string: "Herfst"
SeizoenType volgend = s + 1;       // rekenkundige operatie op enum: Winter

Opgave

  1. Maak in Exercises/Enums een enum Prioriteit met de waarden: Laag, Normaal, Hoog, Kritiek.
  2. Test de volgende conversies:
    • Een enumwaarde tonen als tekst (bv. "Hoog").
    • Dezelfde enumwaarde omzetten naar int.
    • De prioriteit met 1 ophogen via rekenkundige operatie (p1 + 1) en het resultaat tonen.

Verwacht resultaat:

Oefening 5: enum conversies
-------------
Enumwaarde: Hoog
Als int: 2
Nog hogere prioriteit: Kritiek
